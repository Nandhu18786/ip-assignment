My Projects

Music Management System

A music management system is a comprehensive solution designed to organize, manage, and interact with music collections. It allows users to add, delete, and update tracks, organize them by metadata like artist and genre, and create custom playlists. Key features include search and filter capabilities, playback controls, and a user-friendly interface with album art and track lists. It supports integration with online music services, manages user accounts and preferences, and offers reporting on listening habits. Technologies used may include frontend frameworks like React or Vue, backend solutions using Python, and databases like MySQL, with a focus on scalability, security, and performance.

Story Narration

A story narration system generates and presents engaging narratives by leveraging AI technologies. It uses Cohere to create compelling stories from a single prompt, ensuring rich and coherent storytelling. For visual enhancement, the system generates images using Picsart to complement the narrative, providing a visual representation of key scenes or elements. The system integrates these components into a cohesive presentation, where the generated story is paired with corresponding images to create an immersive experience. This approach combines advanced language models with powerful image generation tools to deliver a dynamic and visually appealing storytelling experience.

Sokoban Solver

A Sokoban solver is a program or algorithm designed to find solutions for Sokoban puzzles, which involve moving boxes to designated target locations within a maze-like warehouse. The solver typically uses algorithms to explore possible moves and find a sequence of actions that lead to solving the puzzle.
